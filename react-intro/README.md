# react-intro

Beginner series of tutorials with Vite.

Série de tutoriais para iniciantes com Vite em: https://www.luiztools.com.br/post/tutorial-de-react-js-para-iniciantes/

## How to Run
1. git clone
2. cd react-intro
3. choose one tutorial part
4. npm install
5. npm run dev
