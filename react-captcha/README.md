# react-captcha
A simple Google reCaptcha V2 example in ReactJS.

## How to Run
1. create your reCaptcha key at http://www.google.com/recaptcha/admin
2. git clone
3. cd react-captcha
4. npm install
5. copy .env.example as .env and fill the configs
6. npm start
