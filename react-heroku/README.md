# reactjs-heroku
A simple ReactJS app with Webpack ready to publish at Heroku

Tutorial completo em: https://www.luiztools.com.br/post/como-publicar-aplicacao-reactjs-na-heroku/

## How to Test
1. git clone
2. cd react-heroku
3. npm install
4. npm run dev
5. Access http://localhost:8080

## How to Build
1. npm run build
2. npme start
